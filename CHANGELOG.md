# Changelog

## [1.1.0] - 2026-06-29

### Added

- **entorno.ps1**: Detecta y cierra entornos WAMP/XAMPP en ejecucion
- **gitup.bat**: Panel de control Git con menu interactivo
- **laravel.ps1**: Panel de control global para proyectos Laravel multi-ruta
- **alias_linux.bat**: Emulador de comandos Linux en cmd via DOSKEY

### Changed

- **Launcher.bat**: Ahora ejecuta correctamente scripts .ps1 via powershell -File
- **README.md**: Actualizado con las 4 nuevas herramientas
- **CHANGELOG.md**: Actualizado a version 1.1.0

## [1.0.0] - 2026-06-29

### Added

- **Launcher.bat**: Lanzador principal con deteccion automatica de herramientas en Tools/
- **Desactivar_Defender.bat**: Desactiva o reactiva la proteccion en tiempo real de Windows Defender
- **LimpiOffice.bat**: Escanea y elimina licencias de Office (KMS o residuales)
- **Restaurar_Menu_Clasico_Win11.bat**: Restaura el menu contextual clasico en Windows 11
- **README.md**: Documentacion completa del proyecto
- **CHANGELOG.md**: Registro de cambios
